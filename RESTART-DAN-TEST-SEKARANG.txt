🔴 LANGKAH WAJIB (IKUTI DENGAN TELITI):

STEP 1: TUTUP BROWSER SEPENUHNYA
---------------------------------
1. Tutup SEMUA tab browser
2. Tutup browser sepenuhnya (X di pojok kanan atas)
3. Pastikan browser benar-benar tertutup

STEP 2: STOP SERVER
--------------------
1. Pergi ke window CMD yang menjalankan server
2. Tekan: Ctrl + C
3. Tunggu sampai server stop
4. Tutup window CMD

STEP 3: START SERVER BARU
--------------------------
1. Double click file: START-SERVER.bat
2. Window CMD baru akan terbuka
3. Tunggu sampai muncul:
   "Laravel development server started: http://127.0.0.1:8000"
4. JANGAN tutup window ini!

STEP 4: BUKA BROWSER BARU
--------------------------
1. Buka browser (Chrome/Firefox/Edge)
2. Tekan Ctrl + Shift + Delete
3. Pilih "All time" atau "Sepanjang waktu"
4. Centang: "Cached images and files"
5. Klik "Clear data" atau "Hapus data"
6. Tunggu selesai

STEP 5: AKSES APLIKASI
-----------------------
1. Ketik di address bar: http://localhost:8000
2. Tekan Enter
3. Halaman login akan muncul
