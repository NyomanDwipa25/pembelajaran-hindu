┌──────────────────────────────────────────────────────────────┐
│  STEP 1: GANTI PHP KE VERSI 8.3+                             │
└──────────────────────────────────────────────────────────────┘

Cara Termudah:
--------------
1. Klik kanan icon Laragon (pojok kanan bawah)
2. Pilih: PHP > Version
3. Pilih: php-8.3.32 (atau versi 8.3+ lainnya)
4. Tunggu 5 detik (Laragon restart otomatis)

Verifikasi:
-----------
Buka CMD:
   php -v

Harus muncul: PHP 8.3.x (bukan 8.1.10)

Jika PHP 8.3 TIDAK ADA di list:
--------------------------------
Option A: Update Laragon ke versi terbaru
   Download: https://laragon.org/download/
   Install → akan include PHP 8.3

Option B: Download PHP 8.3 manual
   1. https://windows.php.net/download/
   2. Download: PHP 8.3.x (NTS x64 Thread Safe)
   3. Extract ke: C:\laragon\bin\php\php-8.3.32\
   4. Restart Laragon
   5. Pilih PHP 8.3 di menu

┌──────────────────────────────────────────────────────────────┐
│  STEP 2: ENABLE MYSQL EXTENSION (DI PHP 8.3)                │
└──────────────────────────────────────────────────────────────┘

⚠️ PENTING: Edit php.ini VERSI PHP 8.3, bukan 8.1!

1. Klik kanan Laragon > PHP > php.ini
   (Ini akan buka php.ini PHP 8.3)

2. Tekan Ctrl + F, cari: extension=pdo_mysql

3. Hapus tanda ; (titik koma) di depannya:
   
   SEBELUM:
   ;extension=pdo_mysql
   
   SESUDAH:
   extension=pdo_mysql

4. Cari lagi (Ctrl + F): extension=mysqli

5. Hapus tanda ; di depannya:
   
   SEBELUM:
   ;extension=mysqli
   
   SESUDAH:
   extension=mysqli

6. SAVE (Ctrl + S)

7. Restart Laragon:
   - Klik kanan > Stop All (tunggu 5 detik)
   - Klik kanan > Start All

Verifikasi:
-----------
Buka CMD:
   php -m | findstr pdo_mysql

Harus muncul: pdo_mysql

┌──────────────────────────────────────────────────────────────┐
│  STEP 3: BUAT DATABASE                                       │
└──────────────────────────────────────────────────────────────┘

1. Klik kanan Laragon > MySQL > HeidiSQL
2. Klik kanan di sidebar > Create new > Database
3. Name: pembelajaran_hindu
4. Collation: utf8mb4_unicode_ci
5. OK

┌──────────────────────────────────────────────────────────────┐
│  STEP 4: JALANKAN MIGRATION                                  │

cd E:\app\Coding\laragon\www\pembelajaran-hindu

🎯 KENAPA HARUS PHP 8.3?

Laravel 11 (dirilis 2024) membutuhkan PHP 8.3 minimum.
Ini karena Laravel 11 pakai fitur PHP baru yang ada di 8.3.

Tidak bisa downgrade Laravel ke 10, karena:
- Sudah terlanjur pakai Laravel 11
- Struktur kode sudah pakai fitur Laravel 11
- Butuh rebuild dari awal (4-5 jam lagi)

Jadi solusi terbaik: UPGRADE PHP ke 8.3! ✅
