💡 PENJELASAN TEKNIS:

KENAPA BODY BERGESER KE KANAN?
-------------------------------
Saat modal dibuka, Bootstrap:
1. Menambah class "modal-open" ke body
2. Set body overflow: hidden (disable scroll)
3. Set body padding-right: 15px (kompensasi scrollbar)

Saat modal ditutup, Bootstrap SEHARUSNYA:
1. Remove class "modal-open"
2. Reset overflow
3. Reset padding-right ← INI SERING GAGAL!

Jika padding-right tidak di-reset, body tetap punya padding 15px
di kanan, sehingga konten bergeser ke kiri (tampak di kanan ada space).

FIX SAYA:
---------
Paksa reset semua style body:
- body.style.overflow = ''
- body.style.paddingRight = '' ← INI YANG PENTING!
- body.classList.remove('modal-open')

Dengan 3 baris ini, body kembali normal 100%!
