⚡ LANGKAH RESTART (3 MENIT):

STEP 1: Stop Server
--------------------
Jika server sedang jalan:
1. Pergi ke window CMD yang menjalankan server
2. Tekan: Ctrl + C
3. Ketik: Y (jika ditanya)
4. Tunggu sampai server stop

STEP 2: Restart Server
-----------------------
1. Double click file: START-SERVER.bat
2. Tunggu sampai muncul:
   "Starting development server: http://127.0.0.1:8000"
3. Server siap!

STEP 3: Hard Refresh Browser
-----------------------------
Buka browser dan tekan salah satu:
• Windows: Ctrl + F5
• Atau: Ctrl + Shift + R
• Atau: Clear cache (Ctrl + Shift + Delete)

STEP 4: Test Modal
------------------
1. Login sebagai guru: guru@hindu.com / password
